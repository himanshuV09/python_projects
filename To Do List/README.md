# To-Do List App

This is a simple To-Do List app built with Python using `tkinter` for the user interface and `sqlite3` for data storage.

## Features

- Add tasks
- View tasks
- Delete tasks
- Save tasks to database

## Requirements

- Python 3.8.10
- `tkinter`
- `sqlite3`
